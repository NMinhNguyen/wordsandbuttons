Hello, world!

This is Words and Buttons Online — a growing collection of interactive tutorials, guides and quizzes about things generally considered boring. Maths, software engineering, programming languages.

Maths

  • Programmer's guide to polynomials and splines. An interactive introduction to the curvy things.
  • Mathematical analysis explained with Python, blood, and TNT. A bite-size taste of what is mathematical analysis, and what is it good for.
  • Interactive introduction to iterative algorithms. It shows the very basic concepts on an interactive plot.
  • Interactive guide to homogeneous coordinates. Now with interactive plots. You can play with different kind of transformations and meditate on the matrix.

Software engineering

  • Outperforming everything with anything. Python? Sure, why not? 100 lines of Python that generate LLVM code outperforming C and C++ solutions.
  • Vastly outperforming LAPACK with C++ metaprogramming One trick is enough to help compiler help you.
  • Outperforming LAPACK with C metaprogramming A pair of tricks to help compiler help you.
  • Challenge your performance intuition with C++ magic squares. A quiz with sliders, and a follow-up C++ magic squares demystified with Valgrind and disassembly.

Programming languages

  • You don't have to learn assembly to read disassembly. Three examples of how the deduction alone is enough to get some useful information from the code.
  • Can you tell an assembly language when you see one? An assembly quizz.
  • Learn you a lisp in 0 minutes. A short test that shows that you already know some Lisp even if you don't know that you do.
  • So you think you know C?  Another short C quiz with non-trivial questions and trivial answers.
