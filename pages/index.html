Hello, world!

This is Words and Buttons Online — a growing collection of interactive tutorials, guides and quizzes about things generally considered boring. Maths, software engineering, programming languages.

This is also an attempt to mobilize interactivity as a tool for knowledge sharing. A learning experience not only for our visitors, but for our authors as well.

We already have several things published:

  • Mathematical analysis explained with Python, blood, and TNT. A bite-size taste of what is mathematical analysis, and what is it good for.
  • You don't have to learn assembly to read disassembly. Three examples of how the deduction alone is enough to get some useful information from the code.
  • Interactive introduction to iterative algorithms. It shows the very basic concepts on an interactive plot.
  • Challenge your performance intuition with C++ magic squares, and a follow-up C++ magic squares demystified with Valgrind and disassembly.
  • Interactive guide to homogeneous coordinates. Now with interactive plots. You can play with different kind of transformations and meditate on the matrix.
  • Learn you a lisp in 0 minutes. It's a small test that shows that you already know some Lisp even if you don't know that you do.
  • So you think you know C?  This is a C test with non-trivial questions and trivial answers.

We also have these planned:

  • Сan you tell an assembly code when you see one?
  • Interactive introduction to polynomial functions and splines.
  • Linker slot machine. Winning jackpot with wrongly placed inline.
  • Interactive guide to logarithmic and arctangent scales — how to see literally everything (although in bad detail).
  • Learn stack oriented languages with a pump shotgun.
  • Linear algebra crash course in 12 Python one-liners.

And it is not a finite process, the ideas pop-up all the time.
